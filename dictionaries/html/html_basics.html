Основы

Заголовки

HTML-заголовки определяются тегами <h1> - <h6>.

Тег <h1> соответствует наиболее важному заголовку на странице, а <h6> - наименее важному:

<h1>Это заголовок 1</h1>
<h2>Это заголовок 2</h2>
<h3>Это заголовок 3</h3>

Это заголовок 1

Это заголовок 2

Это заголовок 3

Абзацы

HTML-абзацы определяются при помощи тега <p>.

<p>Это абзац</p>
<p>А это еще один абзац</p>

Это абзац

А это еще один абзац

Ссылки

HTML-ссылки определяются при помощи тега <a>.

<a href="https://webpurple.github.io/learn">Это ссылка</a>

Это ссылка

Адрес ссылки указывается в атрибуте href.

Изображения

Изображение помещается на страницу при помощи тега <img> и ему передаются следующие обязательные атрибуты

  • src - путь к изображению
  • alt - текст, который будет отображаться, если изображение не найдено
<img src="webpurple_logo.png" alt="WebPurple logo">

Но если бы файл отсутствовал...

Кнопки

HTML-кнопки определяются при помощи тега <button>.

<button>Кликни меня</button>

Списки

HTML-списки определяются при помощи тега <ol> (нумерованный список) или <ul> (маркированный список), внутри которых размещены теги <li> (элементы списка).

<ol>
    <li>то</li>
    <li>сё</li>
</ol>
<ul>
    <li>пятое</li>
    <li>десятое</li>
</ul>

  1. то
  2. сё

  • пятое
  • десятое
